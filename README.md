# Backend Proyectos

The app is designed to manage projects. It allows to create and list various types of projects.

***

## Technologies used
* Node.js
* MongoDB
* Postman

***

## System modules
* Tipo Proyecto
* Cliente
* Universidad
* Etapa

***

## Environment variable
create .env with
* MONGO_URI=mongodb+srv://<user>:<password>@cluster1.cyo8srx.mongodb.net/
* PORT=XXXX

***

## Install dependencies
npm install

***

## Run the project
node server
